## Instalação

Para instalar o React execute `$ npm install`

## Execução

Para rodar o servidor execute `$ npm start`

## API

Para realizar a conexão com o Backend vá em `./src/services/api.js` e em `baseURL` e altere para o IP do servidor Backend.
